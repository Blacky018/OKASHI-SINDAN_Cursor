# 開発環境セットアップガイド

## 推奨開発環境

### オプション1: Next.js（推奨）
モダンなReactフレームワークで、SSRや静的サイト生成に対応。デプロイも簡単。

### オプション2: React + Vite
軽量で高速な開発環境。シンプルな構成で始めやすい。

### オプション3: Vue.js + Vite
学習曲線が緩やかで、小規模アプリに適している。

### オプション4: バニラJavaScript
フレームワークを使わず、HTML/CSS/JavaScriptのみで実装。最もシンプル。

---

## 推奨構成：Next.js + TypeScript + Tailwind CSS

### 必要な環境
- Node.js 18以上
- npm または yarn

### セットアップ手順

#### 1. プロジェクトの作成
```bash
npx create-next-app@latest okashi-shindan --typescript --tailwind --app
cd okashi-shindan
```

#### 2. 必要なパッケージのインストール
```bash
npm install
# または
yarn install
```

#### 3. 開発サーバーの起動
```bash
npm run dev
# または
yarn dev
```

ブラウザで `http://localhost:3000` にアクセス

### プロジェクト構成（推奨）
```
okashi-shindan/
├── app/
│   ├── page.tsx          # トップページ
│   ├── diagnose/
│   │   └── page.tsx      # 診断ページ
│   └── result/
│       └── page.tsx      # 結果ページ
├── components/
│   ├── QuestionCard.tsx  # 質問カードコンポーネント
│   ├── ResultCard.tsx    # 結果表示コンポーネント
│   └── Button.tsx        # ボタンコンポーネント
├── data/
│   └── sweets.json       # お菓子データ
├── lib/
│   └── diagnosis.ts      # 診断ロジック
└── public/
    └── images/           # お菓子の画像
```

## 代替案：React + Vite

### セットアップ手順

#### 1. プロジェクトの作成
```bash
npm create vite@latest okashi-shindan -- --template react-ts
cd okashi-shindan
npm install
```

#### 2. Tailwind CSSの追加（オプション）
```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

#### 3. 開発サーバーの起動
```bash
npm run dev
```

## 代替案：バニラJavaScript

### セットアップ手順

#### 1. プロジェクト構成の作成
```bash
mkdir okashi-shindan
cd okashi-shindan
mkdir css js data images
```

#### 2. 基本ファイルの作成
- `index.html` - メインHTML
- `css/style.css` - スタイルシート
- `js/main.js` - メインロジック
- `js/diagnosis.js` - 診断ロジック
- `data/sweets.json` - お菓子データ

#### 3. ローカルサーバーの起動
```bash
# Python 3の場合
python3 -m http.server 8000

# Node.jsの場合（http-serverをインストール）
npx http-server
```

## パッケージ管理

### npm vs yarn vs pnpm
- **npm**: Node.jsに標準搭載
- **yarn**: 高速で安定
- **pnpm**: ディスク容量を節約

どれを使っても問題ありませんが、プロジェクト全体で統一することが重要です。
